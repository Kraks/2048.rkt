2048.rkt
===

A text-based 2048 game written in Racket. 

I wrote a 2048.py before, and doing this in Racket is really just for fun.

Type `w` and press enter for move up, `a` for move left, `s` for move down, and `d` for move right.

Type `q` for quit.
